#### Question

Business analysts need to remove duplicates, standardize column names, merge queries, and load curated data into a Fabric Warehouse. They prefer a visual Power Query experience and do not want to maintain code. Which Fabric item should form the transformation step?

#### Choices

- A. Dataflow Gen2
- B. Fabric notebook
- C. Eventstream
- D. Deployment pipeline

<details>
<summary>Answer and explanation</summary>

#### Correct answer

**A. Dataflow Gen2**

#### Explanation

Dataflow Gen2 provides the visual, low-code Power Query experience described in the scenario. It supports cleansing, derived columns, aggregation, and query merging, and can write results to a Lakehouse or Warehouse.

#### Why the other choices are incorrect

- **A:** Dataflow Gen2 directly meets both the low-code authoring and batch-transformation requirements.
- **B:** A notebook is appropriate for advanced code-first transformations, but it conflicts with the analysts' no-code requirement.
- **C:** Eventstream processes data in motion; the scenario calls for analyst-authored batch preparation with Power Query.
- **D:** A deployment pipeline promotes Fabric content between lifecycle stages; it does not perform the requested data cleansing.
