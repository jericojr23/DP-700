#### Question

An analytics team is building curated fact and dimension tables for enterprise reporting. The team needs SQL-first development, stored procedures, views, multi-table transactional behavior, and a governed semantic model for Power BI. Which Fabric item should host the model?

#### Choices

- A. Warehouse
- B. Eventstream
- C. Shortcut
- D. Dataflow Gen2

<details>
<summary>Answer and explanation</summary>

#### Correct answer

**A. Warehouse**

#### Explanation

Fabric Data Warehouse is designed for enterprise-scale relational analytics on a lake foundation. It is primarily developed with T-SQL and is well suited for star and snowflake schemas, curated data marts, governed semantic models, stored procedures, views, and multi-table ACID transaction support.

#### Why the other choices are incorrect

- **A:** Warehouse matches the SQL-first dimensional modeling, governed reporting, and transactional requirements.
- **B:** Eventstream ingests and routes streaming events; it does not host a dimensional warehouse model.
- **C:** A shortcut references existing data without copying it, but it is not a SQL-first modeling item.
- **D:** Dataflow Gen2 prepares data visually, but it does not provide the relational warehouse surface requested.
