#### Question

A BI team receives weekly CSV extracts from a sales system. The team needs to remove duplicate rows, rename columns, merge lookup data, and load curated output to a gold table. The solution should use a visual Power Query experience and avoid custom code. Which Fabric item should you choose?

#### Choices

- A. Dataflow Gen2
- B. Fabric notebook
- C. Eventstream
- D. Eventhouse

<details>
<summary>Answer and explanation</summary>

#### Correct answer

**A. Dataflow Gen2**

#### Explanation

Dataflow Gen2 is the Fabric low-code data preparation option that uses the Power Query experience. It supports visual transformations such as filters, derived columns, aggregations, and merges, and can write cleaned results to lakehouse or warehouse tables.

#### Why the other choices are incorrect

- **A:** Dataflow Gen2 satisfies the low-code, Power Query, and batch data-cleaning requirements.
- **B:** A notebook is better for code-first transformations and custom Spark logic, which the scenario explicitly avoids.
- **C:** Eventstream is for real-time event ingestion, transformation, and routing rather than weekly file cleanup.
- **D:** Eventhouse is optimized for storing and querying event-based telemetry and log data, not visual batch preparation.
